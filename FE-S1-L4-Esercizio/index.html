COME CREARE UN TEMPLATE PER I POST


di Mario Rossi - pubblicato il 25-10-2020


La prima cosa a fare è suddividere in blocchi logici la pagina, indentificando le parti comuni da applicare a tutte le pagine.


Le varie aree saranno quindi suddivise in:


  • Contenuti di instestazione del tag <header>;
  • Cotenuti centrali variabili del tag <main>(corpo principali)
  • Contenuti di fine pagina comuni,da inserire nel tag <footer>

Formatazzione del codice Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con i css. Perciò i passi saranno:

  1. Progettare su carta il markup finale, assegnango ad ogni specifico testo il tag appropriato
  2. Scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
  3. Applicare gli stili CSS

L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffurzione pressochè totale.
